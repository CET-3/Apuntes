Conceptos básicos de CSS

CSS es el lenguaje que usamos para dar estilo a una página web. Con CSS cambiamos colores, tamaños, espacios, bordes y la forma en que se ve el contenido.

1. ¿Qué es CSS?

CSS significa Cascading Style Sheets. Sirve para definir la apariencia de los elementos que escribimos en HTML.

Color

Permite cambiar el color del texto o del fondo.

Tamaño

Permite modificar el tamaño de letras, imágenes o cajas.

Ubicación

Permite ordenar y acomodar los elementos en la página.

2. ¿Cómo se escribe una regla CSS?

Una regla CSS suele tener un selector y un bloque de propiedades.

p {
  color: blue;
  font-size: 18px;
}

Selector

Indica a qué elemento queremos darle estilo, por ejemplo p o h1.

Propiedad

Es la característica que queremos cambiar, por ejemplo color.

Valor

Es el dato que le damos a la propiedad, por ejemplo blue o 18px.

3. Formas de aplicar CSS

En línea

Se escribe dentro de la etiqueta HTML usando el atributo style.

<p style="color: red;">Hola</p>

Interno

Se escribe dentro de una etiqueta <style> en el archivo HTML.

Externo

Se guarda en un archivo aparte, como assets/css/style.css, y se enlaza desde HTML.

<link rel="stylesheet" href="assets/css/style.css">

En proyectos reales, lo más ordenado suele ser usar un archivo CSS externo.

4. Selectores básicos

Por etiqueta

h1 {
  color: green;
}

Por clase

.destacado {
  background: yellow;
}

Por id

#titulo-principal {
  text-align: center;
}

5. Propiedades muy usadas

Texto

color, font-size, font-family, text-align

Cajas

width, height, padding, margin, border

Fondos

background y background-color

6. Ejemplo completo

h1 {
  color: darkblue;
  text-align: center;
}

p {
  font-size: 18px;
  color: #333333;
}

.tarjeta {
  background: white;
  border: 1px solid gray;
  padding: 16px;
}

Idea simple: HTML arma la estructura y CSS se encarga de cómo se ve.

7. Requisitos

Para practicar CSS, primero hace falta tener un archivo HTML y un archivo CSS enlazados.

Archivo HTML

Es el archivo donde está el contenido de la página.

Archivo CSS

Es el archivo donde se escriben los estilos, por ejemplo assets/css/style.css.

Enlace entre archivos

<link rel="stylesheet" href="assets/css/style.css">
