Conceptos básicos de HTML

HTML es el lenguaje que usamos para darle estructura al contenido de una página web. Con etiquetas definimos títulos, párrafos, listas, enlaces, imágenes y formularios.

HTML básico Introducción VS Code Git y GitHub

1. Estructura general

Un documento HTML suele empezar con estas partes principales:

<!DOCTYPE html>
<html>
  <head>
    <title>Mi página</title>
  </head>
  <body>
    Contenido visible
  </body>
</html>

<head> guarda información para el navegador y <body> contiene lo que ve el usuario.

7. Rutas relativas, absolutas y archivos de la web

Cuando enlazamos una página, una imagen o un archivo, tenemos que indicar dónde está. Eso se hace con una ruta.

Ruta relativa

Apunta a un archivo teniendo en cuenta desde qué carpeta estamos parados.

git-github.html
html5.svg
imagenes/logo.png
../otra-carpeta/pagina.html

Ruta absoluta

Indica la ubicación completa de un archivo, sin depender de dónde estamos parados.

/imagenes/logo.png
C:/proyecto/imagenes/logo.png

Archivo de la web

Es un recurso que se carga desde internet usando una URL completa con https://.

<img src="https://sitio.com/imagen.png" alt="Ejemplo">
<a href="git-github.html">Otra página</a>
<img src="html5.svg" alt="Logo">

Regla simple: una ruta relativa depende de dónde estás parado. Una ruta absoluta indica una ubicación completa. Un archivo de la web se trae desde internet con una URL.

8. Diferencia entre div, span, section y article

div

Es un contenedor genérico de bloque. Agrupa contenido, pero no indica qué tipo de contenido es.

span

Es un contenedor genérico en línea. Se usa para marcar una parte chica de texto.

section

Se usa para agrupar contenido relacionado dentro de una misma sección o tema.

article

Representa una pieza de contenido independiente, como una noticia, una tarjeta o una publicación.

<section>
  <h2>Noticias</h2>

  <article>
    <h3>Nuevo curso de HTML</h3>
    <p>Empieza la semana próxima.</p>
  </article>
</section>

Noticias

Nuevo curso de HTML

Empieza la semana próxima.

<div class="tarjeta">
  <p>Aprendiendo <span class="destacado-texto">HTML</span> paso a paso.</p>
</div>

Aprendiendo HTML paso a paso.

Regla simple: div y span agrupan sin significado. section y article agrupan y además describen qué tipo de contenido es.

9. Por qué a veces aparece &lt; y &gt;

Cuando queremos mostrar código HTML en la pantalla sin que el navegador lo interprete, usamos caracteres especiales.

&lt;

Representa el símbolo <

&gt;

Representa el símbolo >

<p>Hola</p>

Si escribimos <p> como código real, el navegador crea un párrafo. Si escribimos &lt;p&gt;, el navegador lo muestra en pantalla como texto.
