Conceptos basicos de HTML

HTML es el lenguaje que usamos para darle estructura al contenido de una pagina web. Con etiquetas definimos titulos, parrafos, listas, enlaces, imagenes y formularios.

1. Estructura general

Un documento HTML suele empezar con estas partes principales:

<!DOCTYPE html>
<html>
  <head>
    <title>Mi pagina</title>
  </head>
  <body>
    Contenido visible
  </body>
</html>

<head> guarda informacion para el navegador y <body> contiene lo que ve el usuario.

2. Etiquetas basicas

Titulos

Se usan con <h1> hasta <h6>.

Parrafos

Se escriben con la etiqueta <p>.

Contenedores

<div> y <section> ayudan a organizar el contenido.
