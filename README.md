# Apuntes MutuApp

Sitio estático con apuntes introductorios para trabajar HTML, CSS, arquitectura web, VS Code, Git, GitHub y Vercel.

## Contenido

- `index.html`: apuntes básicos de HTML
- `css.html`: apuntes básicos de CSS
- `arquitectura-web.html`: apuntes sobre internet y arquitectura de aplicaciones web
- `herramientas.html`: introducción a VS Code, GitHub y Vercel
- `vscode.html`: apuntes sobre VS Code
- `git.html`: apuntes sobre Git, GitHub y Vercel

```text
propuesta1/
  index.html
  css.html
  arquitectura-web.html
  herramientas.html
  vscode.html
  git.html
  README.md
  assets/
    css/
      style.css
    img/
      html5.svg
      logo-unidos-cet3.png
```

## Navegación

Las páginas están conectadas entre sí mediante un menú superior:

- Arquitectura web
- HTML básico
- CSS básico
- Herramientas
- VS Code
- Git y GitHub
