# Apuntes MutuApp

Sitio estático con apuntes introductorios para trabajar HTML, CSS, VS Code, Git, GitHub y Vercel.

## Contenido

- `index.html`: apuntes básicos de HTML
- `css.html`: apuntes básicos de CSS
- `herramientas.html`: introducción a VS Code, GitHub y Vercel
- `vscode.html`: apuntes sobre VS Code
- `git.html`: apuntes sobre Git, GitHub y Vercel

## Estructura

```text
assets/
  css/
    style.css
  img/
    html5.svg
    logo-unidos-cet3.png
```

## Cómo verlo

Podés abrir `index.html` directamente en el navegador o usar una extensión como Live Server en VS Code.
