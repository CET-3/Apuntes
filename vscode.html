VS Code

Este apunte explica para qué sirve VS Code y por qué es mejor usar un editor de código que un editor de texto común cuando estamos aprendiendo a programar.

1. ¿Qué es VS Code?

VS Code es un editor de código. Sirve para escribir, ordenar y revisar archivos de un proyecto. También permite trabajar con varias carpetas, extensiones y herramientas útiles para programar.

Escritura

Permite escribir HTML, CSS, JavaScript y muchos otros lenguajes.

Organización

Muestra los archivos y carpetas del proyecto de una forma clara.

Herramientas

Incluye búsqueda, terminal integrada y conexión con Git.

2. ¿Por qué usar un editor de código?

Un editor de código está pensado para programar. No solo deja escribir, sino que también ayuda a entender y organizar mejor el trabajo.

Ayuda visual

Colorea etiquetas, muestra mejor el código y hace más fácil encontrar errores.

Más orden

Permite trabajar con carpetas, archivos y sangrías de una manera más clara.

Herramientas útiles

Puede autocompletar, marcar errores y conectarse con Git y GitHub.

Idea simple: un editor de texto sirve para escribir. Un editor de código sirve para escribir código mejor.

3. Partes básicas de la interfaz

Explorador

Está a la izquierda y muestra los archivos y carpetas del proyecto.

Editor

Es la parte central donde escribimos y modificamos el código.

Terminal

Permite ejecutar comandos sin salir del editor.

Source Control

Sirve para ver cambios del proyecto y trabajar con Git desde la interfaz.

4. Qué cosas nos ayuda a hacer

  1. Abrir un proyecto completo.
  2. Editar varios archivos a la vez.
  3. Buscar palabras en todo el proyecto.
  4. Ver errores o advertencias mientras escribimos.
  5. Trabajar con Git y GitHub desde botones y menús.

5. Extensiones útiles

VS Code permite instalar extensiones. Las extensiones agregan funciones nuevas al editor.

Live Server

Sirve para abrir una página HTML en el navegador y ver los cambios de forma más rápida.

Ver extensión Live Server

Codex

Sirve para recibir ayuda con código desde VS Code usando una cuenta de ChatGPT.

Más información sobre Codex

Según la ayuda oficial de OpenAI, por el momento Codex también está disponible para cuentas ChatGPT Free, aunque con límites de uso y sujeto a cambios.

6. Relación con GitHub

VS Code no reemplaza a GitHub. Son herramientas distintas que trabajan juntas.

VS Code

Se usa para crear y modificar el proyecto.

GitHub

Se usa para guardar el proyecto en internet y compartirlo.

Idea simple: en VS Code escribimos el proyecto. En GitHub lo guardamos y compartimos.
